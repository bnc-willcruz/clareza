# Compass.ia

Diagnóstico interativo que responde a uma pergunta prática: **vale usar IA (ChatGPT, Claude) neste caso de uso — e como?**

O usuário descreve um caso de uso do dia a dia e responde 8 perguntas que avaliam duas dimensões:
- **Afinidade da tarefa com IA** (a IA *consegue* fazer bem?)
- **Segurança para usar IA** (a IA *deve* ser usada, considerando os riscos?)

O resultado classifica o caso de uso e dá recomendações específicas, operacionalizando um framework de prontidão para IA em uma ferramenta de 2 minutos.

**Stack:** React + Tailwind + lucide-react.

> Observação: o arquivo `.tsx` é o código-fonte do componente. A versão funcional roda como aplicação React.
