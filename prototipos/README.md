# Protótipos

Protótipos navegáveis. Os arquivos `.html` abrem direto no navegador (baixe e clique duas vezes).

| Arquivo | O que é |
|---------|---------|
| `plataforma-conceito.html` | Plano de negócios completo navegável + protótipo da plataforma Clareza (dashboard, chat, B2B) |
| `piloto-dolcevita.html` | Protótipo do produto com a identidade real do piloto: dashboard, pedidos, financeiro, Instagram, chat |
| `demo-fluxo-magico.html` | Demo de 60s: conversa de WhatsApp virando pedido no painel ao vivo |
| `conceito-generico.html` | Versão genérica do conceito (antes da marca do piloto) |
| `formulario-semanal.html` | Formulário de check-in semanal (captura de contexto de baixo atrito) |
| `simulador-mercado.jsx` | Simulador interativo de mercado, unit economics e investimento (React) |

> Nota: os protótipos usam dados ilustrativos para demonstração. O sistema em produção (`apps-script/`) usa dados reais.
